# Teste MicroData API

## 📌 Descrição
Projeto de estudo em .NET/C# criado para testar a implementação de uma API chamada **MicroDataAPI**.  
Este repositório representa um experimento inicial e não possui aplicação prática em produção.

## 🛠️ Tecnologias Utilizadas
- .NET / C#
- Visual Studio

## 🎯 Objetivo
Explorar conceitos básicos de criação de APIs em .NET e realizar testes com dados estruturados.

## 📂 Estrutura
- **MicroDataAPI** → Projeto principal em C#.
- **.vs** → Arquivos de configuração do Visual Studio.
